Create a virtual enivonrment:
1. pip install virtualenv
2. virtualenv venv

3. Mas/OS - source venv/bin/activate
4. Windows - venv\Scripts\activate

5. pip install -r requirements.txt

Run Code:
6. In command line type python3 flaskapp.py
